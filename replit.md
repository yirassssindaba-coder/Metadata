# File Metadata Microservice

## Overview
A freeCodeCamp API project that provides a file metadata microservice. Users can upload a file and receive JSON metadata including the file name, type, and size.

## Project Structure
- `index.js` - Main Express server
- `views/index.html` - Frontend HTML page with file upload form
- `public/style.css` - Stylesheet
- `package.json` - Node.js dependencies

## Tech Stack
- Node.js with Express
- CORS enabled
- dotenv for environment configuration

## Running the Application
The app runs on port 5000 via `npm start`.
